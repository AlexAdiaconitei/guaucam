GuauCam turns a Raspberry Pi Zero W and a camera (USB webcam or CSI module) into a private dashboard with live video, a noise meter and Telegram alerts when your dogs bark too much.

Install on the Pi Support on Ko-fi
-24.0 dB

The yellow mark is your threshold. If the bar stays above it for several seconds, you get an alert.

Video without recompression

The USB webcam outputs MJPEG and the Pi forwards it; with a CSI module, the hardware encoder compresses it. The CPU never transcodes, which matters on a Zero W.

Audio stays at home

The microphone is only measured for volume. It is not transmitted, not stored and has no history.

Useful alerts

If noise crosses the threshold for the configured time, Telegram receives a message and photo.

Private access

Use it through Tailscale or your LAN. Nothing is exposed to the internet and no router ports are opened.

How it works

A deliberately simple flow: the camera does the heavy work, the Pi serves video and measures noise, and your phone only opens a private URL.

USB or CSI cameraC920/C922 webcam (native MJPEG + microphone) or a ribbon module such as Pi Camera.
Streamerustreamer (USB) or picamera2 (CSI) publish the HTTP stream on port 8080.
Noise detectorMeasures dBFS, applies threshold, sustained duration and cooldown.
Web panelVideo, live meter and settings applied immediately.
TelegramAlert with photo when barking runs too long.

Installation

The installer detects your camera (and asks if there are several), sets up systemd services, Tailscale, the streamer and the base configuration. Running it again updates the code without touching /etc/guaucam.conf.

Before you start
Raspberry Pi Zero W v1Raspberry Pi OS Lite 32-bit Bookworm, SSH enabled and WiFi.
USB or CSI cameraUVC webcam with MJPEG (C920/C922) or OV5647/Pi Camera ribbon module. The installer detects what you have.
Tailscale accountApp installed on the phone or PC where you will watch the panel.
Telegram botOnly for alerts. Talk to @BotFather and save the token.
# 1. In Telegram: /newbot with @BotFather
# 2. Send any message to your new bot

curl -fsSL https://github.com/AlexAdiaconitei/guaucam/raw/main/install.sh | sudo bash

# Update later:
sudo bash /opt/guaucam/install.sh

Create the bot

In Telegram, talk to @BotFather, send /newbot and save the token. Then write any message to the bot so the installer can detect your chat_id.

Run the installer

Over SSH, launch the installation command. It downloads dependencies, clones the repo into /opt/guaucam and prepares the services.

Authorize Tailscale

The installer will show a URL. Open it once to add the Pi to your tailnet.

Save Telegram and open the panel

Enter the bot token when asked. At the end you will see the access URLs and receive a test message.

Daily use

With Tailscale active, open these URLs from your phone or PC. They also work with the Tailscale IP instead of the name.

http://<pi-name>:8081/Panel

Video, noise level and detector settings.

http://<pi-name>:8080/Stream

Live video only, without controls.

http://<pi-name>:8080/snapshotSnapshot

Instant capture for alerts or checks.

Calibrate without restarting

The panel adjusts threshold, sustained duration, cooldown and Telegram immediately. The rest of the configuration lives in /etc/guaucam.conf.

First calibration
  1. Open the panel on :8081.
  2. Watch where the bar sits with normal home noise.
  3. Make noise or wait for barking and see how high it rises.
  4. Drag the yellow threshold between both levels and save.
# Service status
systemctl status guaucam-stream guaucam-detector

# Live logs
journalctl -u guaucam-stream -u guaucam-detector -f

# Restart after advanced configuration edits
sudo systemctl restart guaucam-stream guaucam-detector

Clear limits

Designed for live watching and volume alerts, not recording, classifying audio or replacing a commercial cloud camera.

No history

There is no recording. The system is live viewing and one-off alerts.

No remote audio

Audio is measured and discarded. It is never retransmitted or stored.

It does not recognize barks

It measures volume. A vacuum or construction noise can trigger alerts; threshold and duration reduce false positives.

Realistic Pi Zero

If it stutters away from home, drop to 800x600 or 10 FPS. The 2.4 GHz WiFi sets the limit.
